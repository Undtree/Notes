# Intro

在过去，计算机系统往往都是高度中心化的；然而现在，自治的计算机越来越多，这些计算机也需要互联来完成任务，因为我们有了计算机网络和分布式系统。

??? Hint Computer network v.s. Distributed system

    – Computer network（计算机网络）: a collection of interconnected, autonomous computing devices. 
    
    – Distributed systems（分布式系统）: a collection of autonomous computers appears to its users as a single coherent system. Coherence, model, implementation (middleware).
    
    – Examples: The Internet is not a single network but a network of networks. The Web is a distributed system that runs on top the of the Internet.
