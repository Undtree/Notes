## 传统电子取证

现代文件系统删除文件，通常是删除指向文件的文件指针，但往往不从硬盘擦除数据。如果使用“360 强力删除”等手段，它会在原文件存在的位置覆写，使得文件被破坏。对于固态硬盘来说，这种覆写操作更为有效；但对于机械硬盘，在擦写次数不多的情况下，磁道边缘残留的磁效应仍然可以使得原数据被恢复。
